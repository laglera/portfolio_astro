# Beyond Horizons ✈️ — Agencia de Viajes y Aventura

**Beyond Horizons** es una plataforma web moderna, inmersiva y de alto rendimiento visual diseñada para una agencia de viajes de primer nivel. El objetivo principal es inspirar a los viajeros a explorar destinos espectaculares alrededor del mundo y facilitar la planificación y reserva de sus próximas aventuras de manera fluida y elegante.

El proyecto está diseñado bajo estrictos principios de diseño minimalista, tipografía editorial y responsividad absoluta, ofreciendo una experiencia inmersiva desde el primer vistazo.

---

## 🚀 Características Clave

* **Diseño Hero de Impacto:** Una sección de bienvenida inmersiva con fondos de alta resolución, transiciones fluidas y una tipografía majestuosa que invita a la exploración.
* **Buscador de Aventuras:** Un módulo de reserva y búsqueda rápida interactivo para filtrar destinos, fechas y presupuestos de viaje.
* **Catálogo de Destinos Curados:** Cuadrícula responsiva que presenta las mejores aventuras (montaña, playas paradisíacas, safaris y turismo urbano) con efectos de zoom y revelación en hover.
* **Sección de Experiencias (Testimonios):** Carrusel interactivo y minimalista donde los viajeros comparten sus testimonios e imágenes reales de sus expediciones.
* **Formulario de Contacto & Newsletter:** Un módulo de suscripción elegante para recibir guías de viaje y newsletters personalizadas.

---

## 🛠️ Tecnologías y Estándares

El proyecto está construido utilizando código puro y optimizado para garantizar tiempos de carga imperceptibles y una compatibilidad total:

* **HTML5 Semántico:** Estructura robusta, accesible y optimizada para SEO y lectores de pantalla.
* **CSS3 Premium:** Estilos a medida mediante Variables CSS (Custom Properties), Flexbox y CSS Grid para lograr un diseño responsivo sin depender de frameworks pesados.
* **Animaciones y Transiciones de GPU:** Micro-interacciones interactivas, efectos hover con halos luminosos y oleajes dinámicos optimizados para ejecutarse a 60fps constantes.
* **Vanilla JavaScript:** Lógica interactiva ligera (control del carrusel, buscador y animaciones de scroll) libre de dependencias externas.

## 💻 Instalación y Uso Local

Al estar desarrollado en código nativo libre de compiladores, ejecutar el proyecto localmente es sumamente sencillo:

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/laglera/beyondhorizons.git
   ```
2. **Navegar a la carpeta:**
   ```bash
   cd beyondhorizons
   ```
3. **Ejecutar el proyecto:**
   * Simplemente abre el archivo `index.html` en tu navegador web de preferencia, o utiliza la extensión **Live Server** de VS Code para una experiencia de desarrollo en tiempo real.

## 🎨 Principios de Diseño

* **Simplicidad Apple-Style:** Uso generoso del espacio en blanco, tipografías refinadas sans-serif y ausencia de elementos pesados o caóticos.
* **Fotografía Inmersiva:** La composición visual cede el protagonismo a las imágenes de los destinos, logrando una conexión emocional instantánea con el usuario.
* **Interactividad Orgánica:** Cada botón, tarjeta y enlace reacciona de manera predecible y suave al cursor, ofreciendo un feedback de interacción sumamente premium.
